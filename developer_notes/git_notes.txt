**How to rebase and repoint a feature branch on develop branch**


    git checkout **feature-processqa**
    git pull --rebase
    git rebase **develop**
    git push --force-with-lease
    git update-ref -m "reset: Reset **feature-processqa** to **fb0c65fd84d67e16251f20d3452abf212a7cd568**" refs/heads/**feature-processqa** **fb0c65fd84d67e16251f20d3452abf212a7cd568**
    git push --force-with-lease
